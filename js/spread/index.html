Spread syntax

Spread syntax can be used when all elements from an object or array need to be included in a list of some kind.

When we invoke the function, we pass it all the values in the array using the spread syntax and the array name.

It is commonly used when you want to add a new item to a local data store, or display all stored items plus a new addition.

Spread syntax

1. Add the newNumber to the numberStore.


let numberStore = [0, 1, 2];

const newNumber = 43;


numberStore = [...numberStore, newNumber];

console.log(numberStore);



"Array(4) [ 0, 1, 2, 43 ]"

2. Log all elements of the characters array

We pass all elements of the object or array that is going to be itterated as arguments to the function.


function logSpread(...element) {

        console.log(element);

}


logSpread(...characters);



"Array(4) [ {…}, {…}, {…}, {…} ]"

3. Add an ID property with a random ID to each of the characters creating a new charactersWithId array

Any argument in the argument list can use spread syntax, and the spread syntax can be used multiple times.


function addIdToCharacter(element) {

        const newId = { id: Math.floor(Math.random() * 10000) };

        return { ...element, ...newId };

        }


const charactersWithId = characters.map(addIdToCharacter);

console.log(charactersWithId);



"Array(4) [ {…}, {…}, {…}, {…} ]"
